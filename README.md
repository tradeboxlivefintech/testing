# Express Server

A basic Express.js server setup.

## Installation

```bash
npm install
```

## Running the Server

**Production:**
```bash
npm start
```

**Development (with auto-reload):**
```bash
npm run dev
```

The server will start on `http://localhost:3000` (or the port specified in the PORT environment variable).

## Endpoints

- `GET /` - Welcome message
- `GET /health` - Health check endpoint
